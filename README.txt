🎵 Modern Music Player
A responsive and interactive web-based music player built using HTML, CSS, and JavaScript. The player features a sleek modern UI, playlist support, progress tracking, and volume control.

🔍 Features
🎧 Audio Playback: Play, pause, skip forward, and skip back between songs.

🖼️ Album Art: Displays cover art for the currently playing track.

🎼 Song Info: View title, artist, duration, and current playback time.

📶 Progress Bar: Real-time playback progress with click-to-seek functionality.

🔊 Volume Control: Adjustable volume slider with smooth UX.

📜 Playlist: Scrollable list of songs, dynamically updated when selected.

🎨 Modern UI: Clean and minimal design with hover effects and responsive layout.

📌 Notes
This is a frontend-only music player. Audio files are streamed from external URLs.

The current implementation autoplays the first track once the page loads.

Volume is preset to 0.7 on load.
